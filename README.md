# Retail-Sales-Analytics-Project
Built an end-to-end Retail Sales Analytics pipeline using Databricks Lakehouse architecture. Implemented ELT pipelines following the Architecture (Bronze, Silver, Gold) using Delta Lake.
Ingested raw CSV retail data, cleaned and transformed it into analytics-ready datasets, and generated business KPIs such as total revenue, monthly sales, and top-selling products. Enabled scalable analytics using SQL Warehouses and governed data access using Unity Catalog. Designed the solution for performance, reliability, and business consumption.
